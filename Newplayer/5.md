# 每日礼包

### 1.**每日签到和在线任务：**

每天签到和在线可以获得最高270点券的奖励

##### 每日签到：通过菜单或使用指令`/qd`进行签到，连续签到有额外奖励

![签到菜单](../Newplayer/image/签到菜单.png)



**在线任务：**

| 在线时长 | 点券奖励 |
| :------: | :------: |
|  10分钟  |   1-20   |
|  30分钟  |   1-50   |
|  60分钟  |  1-100   |
