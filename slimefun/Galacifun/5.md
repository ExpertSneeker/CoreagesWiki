# 星球环境汇总

更详细的星球信息请前往[星球/天体](celestialboby/0.md)查看

辐射，热，寒冷三个模块，对应的状态效果为：凋零，着火，减速+冻伤。

|    星球     |  环境(所需模块)  |   星球buff    |
| :---------: | :--------------: | :-----------: |
|    月球     |     1级抗寒      | 4 级跳跃提升  |
|    火星     |     2级抗寒      | 2 级跳跃提升  |
|    金星     |     10级耐热     | 0 级跳跃提升  |
| 木卫一-艾奥 | 5级耐热、3级辐射 | 4 级跳跃提升  |
|   木卫二    |     3级抗寒      | 5 级跳跃提升  |
|   土卫二    |     3级抗寒      | 12 级跳跃提升 |
| 土卫六-泰坦 |     6级抗寒      | 5 级跳跃提升  |

注意：土卫六-泰坦

生物**水蛭**，原型为蠹虫 （**泰坦王**也会召唤水蛭）

它在攻击玩家时，有概率吞掉玩家随机一个物品栏的物品，但是它被杀时会掉落所有吞食过的物品。
